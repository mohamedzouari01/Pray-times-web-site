# Pray-times-web-site
A web application that displays daily prayer times (Fajr, Dhuhr, Asr, Maghrib, Isha) by city using the AlAdhan API. Built using HTML, CSS, and JavaScript.
## Features
- Daily prayer times
- City selection
- Uses public API
